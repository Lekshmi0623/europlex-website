# EuroPlex Industrial Website

A modern and fully responsive industrial website designed for showcasing HDPE pipes, coils, fittings, and infrastructure solutions.  
Built using HTML, CSS, and JavaScript with a premium UI/UX approach suitable for industrial and manufacturing companies.

---

## 🌐 Live Demo

🔗 https://lekshmi0623.github.io/europlex-website/

---

## 📌 Project Overview

EuroPlex is a professional industrial company website prototype created to demonstrate modern frontend development concepts including responsive layouts, interactive UI components, premium design styling, and structured content presentation.

The website focuses on delivering a clean and modern user experience for industrial product showcasing and infrastructure solution branding.
